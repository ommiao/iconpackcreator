Dynamic icons probably only support calendars on third-party desktops, and calendars, weather, and clocks on OnePlus desktops. The following explanation will be in order from simple to complex.

First, let's adapt the common dynamic calendar. The adaptation of the common dynamic calendar is divided into two simple steps.

The first step is to collect all the adaptation items that need to be adapted to the dynamic calendar in a group, and name the icon of this group as "Calendar" (not the icon file named "Calendar", the name of the icon file is irrelevant ). For example, I have two calendar adaptation items that need to be adapted to the dynamic calendar, then there are two calendar adaptation items, as shown below:

The second step is to select the icon on the dynamic icon editing page. The common dynamic calendar is composed of 31 icons, and each date will be displayed using the corresponding icon, and if there are not enough 31 icon files, it may not take effect, so be sure to select all the icons when testing.

After selecting the calendar icon as shown below:

After the two-step operation is completed, export apk again and apply it to the desktop to see the effect.

Oneplus dynamic clockis also very simple, only need to select three pictures, namely background, hour hand and minute hand, after selection, as shown below:

The following focuses on Oneplus Dynamic Weather and Oneplus Dynamic Calendar. These two dynamic icons are divided into two parts, one is icon resources and the other is dynamic text.

The icon resource of Oneplus Dynamic Calendar has only one background, and the text shows the date of the day; the icon resource of Oneplus Dynamic Weather is 8 common weathers, and the text is the real-time temperature, which can be displayed or not.

After selecting the icons of Oneplus Dynamic Weather and Calendar, as shown below:

In fact, after completing these operations, re-export the icon pack apk and apply it to the Oneplus launcher you can see the effect. The text style is the default white and centered, which may not match the icon you choose. If you have no effect after applying it, please pay attention to the output log when exporting. The processing steps of the icon will be reflected in detail.
